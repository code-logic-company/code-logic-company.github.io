Cara Merubah atau Menambahkan Port OpenSSH di Debain 9

Mengubah Port OpenSSH


1. Login atau Masuk Server dengan Username default : root

2. Jika sudah login, masuk pada bagian default port OpenSSH yang terletak di /etc/ssh/sshd_config dengan cara nano /etc/ssh/sshd_config

Enter untuk masuk

3. Tampilan :

/etc/ssh/sshd_config

4. Cari tulisan :

Port 22
Port 444
#AddressFamily any
#ListenAddress 0.0.0.0
#ListenAddress ::

5. Lalu ubah Port 22 atau Port 444 dengan yang di inginkan

6. Jika sudah Tekan CTRL + X Lalu Y untuk konfirmasi dan Enter.

7. Masukan perintah :

service ssh restart

Untuk mreubah atau restart Port OpenSSH

8. Selesai.
